# statistical-analysis-study-korea-univ
 
고려대학교 석준희 교수님의 강의안 공부
- EDA과정에서 필요한 두 표본집단 분석, P-value, 상관관계 분석 등 데이터 모델링 이전에 필요한 데이터 탐색 과정을 파이썬의 stats, sklearn 라이브러리를 이용하여 학습

o 학습 내용

- 두 표본집단 분석
- P-value
- 상관관계
- CV(Cross Validation)
- 선형회귀
- 로지스틱 회귀
- 최소제곱의 의미
- 최대우도의 의미 등

o 이용 라이브러리
- Pandas
- numpy
- sklearn
- stats
- matplotlib 등
